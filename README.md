*# 📌 IoT Enabled Automated Object Recognition for the Visually Impaired

## 📖 Overview
This project is an **IoT-based assistive system** designed to help visually impaired individuals navigate their environment independently. It uses **Machine Learning, Deep Learning, and IoT technologies** to detect objects, recognize currency, and provide real-time audio feedback.

The system improves **mobility, safety, and independence** by enabling users to understand their surroundings without relying on others.

---

## 🎯 Features
- 🔍 Real-time object detection using Deep Learning  
- 💵 Currency recognition for financial transactions  
- 📡 IoT integration for remote monitoring and data transmission  
- 🎧 Audio feedback for detected objects and obstacles  
- 🚧 Multi-directional obstacle detection (front, left, right, ground)  
- 📱 Portable and user-friendly design

## 🏗️ System Architecture
The system consists of the following modules:

### 1. Object Detection Module
- Detects and classifies objects using SSD + MobileNet  

### 2. Obstacle Detection Module
- Uses sensors to identify obstacles in multiple directions  

### 3. Currency Recognition Module
- Identifies different currency denominations  

### 4. Audio Output Module
- Converts detection results into speech  

### 5. IoT Module
- Sends data to a remote server for monitoring  

### 6. Database Module
- Stores user data, detection logs, and location history
